# 日本史年表作成・公開ツール

HTML / CSS / JavaScript だけで動作する静的な年表サイトです。  
GitHub Pages などの静的ホスティングでそのまま公開できます。

## 概要

- **公開ページ**（`index.html`）: 年表を閲覧専用で表示します。編集UIは表示されません
- **編集ページ**（`edit.html`）: ブラウザ上で年表データの作成・編集・エクスポートができます
- 西暦↔和暦の自動変換機能付き
- 作成したデータを JSON としてエクスポートし、公開用データとして利用できます
- サーバーサイド処理は一切不要です

## ファイル構成

```
timeline-site/
├─ index.html          … 公開用ページ（閲覧専用）
├─ edit.html           … 編集用ページ（管理者向け）
├─ style.css           … スタイルシート（共通）
├─ viewer.js           … 公開ページ用スクリプト
├─ app.js              … 編集ページ用スクリプト
├─ era-data.js         … 元号データ（応仁〜令和）
├─ timeline-data.json  … 公開用年表データ（JSON）
└─ README.md           … このファイル
```

## ローカルでの起動方法

`timeline-data.json` を `fetch` で読み込むため、`file://` プロトコルでは正常に動作しません。  
ローカルサーバーを起動してください。

### Python がある場合

```bash
cd timeline-site
python -m http.server 8000
```

- 公開ページ: http://localhost:8000
- 編集ページ: http://localhost:8000/edit.html

### Node.js がある場合

```bash
cd timeline-site
npx -y serve .
```

表示された URL をブラウザで開いてください。

### VS Code を使っている場合

拡張機能「Live Server」をインストールし、`index.html` を右クリック → 「Open with Live Server」で起動できます。

## GitHub Pages での公開方法

1. GitHub にリポジトリを作成します
2. `timeline-site/` 内のファイルをリポジトリのルートにプッシュします
3. リポジトリの Settings → Pages で、ソースを `main` ブランチの `/ (root)` に設定します
4. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます

> **💡 ヒント**  
> `edit.html` は公開サイトにも含まれますが、パスワード保護などはありません。  
> 編集ページへのリンクをサイト上に設置しなければ、閲覧者が直接アクセスすることは稀です。  
> 必要に応じて `edit.html` を `.gitignore` に追加し、公開リポジトリから除外することもできます。

## 年表データの更新方法

### 方法1: ブラウザで編集してエクスポート

1. `edit.html` を開き、フォームから年表データを追加・編集します
2. 「JSONエクスポート」セクションで「エクスポート」ボタンを押します
3. 表示された JSON をコピーします
4. `timeline-data.json` の内容を上書きします
5. 変更をコミット・プッシュして公開サイトに反映します

### 方法2: JSON ファイルを直接編集

`timeline-data.json` を直接テキストエディタで編集することもできます。  
データ形式については下記を参照してください。

## JSON データ形式

```json
[
  {
    "id": "一意のID",
    "western": {
      "year": 1582,
      "month": 6,
      "day": 2,
      "display": "1582年6月2日"
    },
    "japanese": {
      "era": "天正",
      "year": 10,
      "month": 6,
      "day": 2,
      "display": "天正10年6月2日"
    },
    "event": "出来事の内容",
    "note": "備考（任意）",
    "sortKey": "1582-06-02"
  }
]
```

### sortKey のルール

| 情報 | sortKey 形式 |
|------|-------------|
| 年のみ | `1582-00-00` |
| 年月のみ | `1582-06-00` |
| 年月日 | `1582-06-02` |

## JSONエクスポートと timeline-data.json の差し替え方法

1. ブラウザで `edit.html` を開きます
2. 年表データを追加・編集します
3. ページ下部の「JSONエクスポート」セクションで「エクスポート」をクリックします
4. 「JSONをコピー」ボタンでクリップボードにコピーします
5. `timeline-data.json` をテキストエディタで開き、全内容を貼り付けて保存します
6. Git でコミット・プッシュすれば公開サイトに反映されます

## ブラウザ内保存について

- 編集中のデータは `localStorage` に自動保存されます
- ページを閉じても、次回アクセス時にデータを復元できます
- ただし、これはブラウザ内のローカル保存であり、公開サイトの `timeline-data.json` は更新されません
- 公開データを更新するには、上記のエクスポート手順で JSON を差し替えてください

## 旧暦・新暦変換について

> **⚠️ 注意**  
> 現在のバージョンでは、旧暦から新暦への厳密な変換は実装されていません。  
> 明治以前の日付について、月日は入力値をそのまま対応させています。  
> 例：「天正10年6月2日」→「1582年6月2日」として扱われます。  
> 実際の新暦日付とは異なる場合があります。
